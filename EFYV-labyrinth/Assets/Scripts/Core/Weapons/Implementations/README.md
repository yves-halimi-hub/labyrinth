Configured concrete weapons grouped by attack family:

- [`AuraImplementations.cs`](AuraImplementations.cs): Garlic and Spinning Swords.
- [`DropImplementations.cs`](DropImplementations.cs): Bomb and Meteor.
- [`MeleeImplementations.cs`](MeleeImplementations.cs): Aluminum Bat and Longsword.
- [`OrbitalImplementations.cs`](OrbitalImplementations.cs): Spinning Axe and Beyblade.
- [`SplashImplementations.cs`](SplashImplementations.cs): Lightning and Holy Water.

Defaults come from the shared game configuration. Keep values centralized there so gameplay, tests, and tooling use the same contract.
