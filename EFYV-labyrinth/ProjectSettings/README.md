# Unity Project Settings

[Back to the Unity project](../README.md)

These serialized Unity settings pin editor version `6000.3.20f1`, build scenes, input, tags/layers,
physics, graphics, quality, time, memory, version control, and service behavior. They are authored
project configuration, not local Unity `Library` state. Changes require the static asset suite and a
Unity Test Framework/editor run on the pinned version.
