# LabyMake Application Source

[Back to LabyMake](../README.md)

`app.efyv` is the sole entrypoint. It composes the generic Maker studio from [maker](maker/README.md),
the [studio page](pages/README.md), the [engine integration](integrations/README.md), and the
[access boundary](config/README.md). Compiler reachability—not directory scanning—decides which
declarations participate in the application plan.
